# 文字数・バイト数カウンター

テキストを入力すると、リアルタイムで **文字数** / **空白除く文字数** / **行数** / **段落数** / **UTF-8 バイト数** を表示するシングルページアプリ（単一 HTML ファイル）です。

---

## 特長

* **単一ファイル**：`index.html` だけで動作。サーバーやビルド不要。
* **リアルタイム表示**：入力と同時に数値が更新。
* **絵文字・合成文字対応**：`Intl.Segmenter` を用いたグラフェム単位のカウント。
* **モダンUI**：ダーク/ライト両対応、視認性に配慮。
* **ローカル保存**：入力内容はブラウザの `localStorage` に自動保存。

---

## 画面の見方

* **文字数**：見た目の「1文字」を数えます（絵文字・合成文字も 1 として扱います）。
* **空白除く文字数**：スペース/改行などの空白を取り除いて数えます。
* **行数**：改行で区切られた行の数（CR/LF いずれも対応）。
* **段落数**：1 行以上の空行で区切られたブロック数。空行がなければ 1 と数えます。
* **バイト数（UTF-8）**：`TextEncoder` で UTF-8 に変換したときの長さ（バイト）。

## よくある質問（FAQ）

**Q. 絵文字はどう数えていますか？**
A. ブラウザの `Intl.Segmenter('ja', { granularity: 'grapheme' })` が使える場合はそれを利用し、使えない環境ではコードポイントで数えます。

**Q. 段落の定義は？**
A. 1 行以上の空行で区切られたテキストのまとまりを 1 段落とします。

**Q. バイト数はなぜ必要？**
A. 入力サイズの上限があるフォームや API、文字数制限のある場面で目安になります。

## 技術メモ（簡潔）

* 文字数：グラフェム単位（`Intl.Segmenter`）でカウント。フォールバックあり。
* 空白除去：半角/全角スペース、タブ、各種改行・区切り空白をまとめて除去。
* 行・段落：正規表現で CR/LF や空行を判定。
* バイト数：`TextEncoder().encode(text).length`。
* 自動保存：`localStorage` に `counter:text` キーで保存。
